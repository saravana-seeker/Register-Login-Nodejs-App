# Setup 
```
git clone https://github.com/saravana-seeker/Register-Login-Nodejs-App.git
cd Register-Login-Nodejs-App
```
# Create .env with following content
```

Jwt_Key=Jwttokensecretkey
```

# NPM

```
npm build
npm start
```
